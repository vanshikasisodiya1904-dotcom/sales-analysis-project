# Sales Data Analysis Dashboard

##  Overview
This project performs exploratory data analysis (EDA) on a sales dataset to extract meaningful business insights. It includes data cleaning, analysis, and visualization of sales performance across products and categories.

---

##  Features
- Data cleaning and preprocessing
- Analysis of total sales and profit
- Identification of top-selling products
- Category-wise performance analysis
- Sales trend visualization over time

---

##  Tech Stack
- Python 
- Pandas
- Matplotlib
- Seaborn

---

## Dataset
The dataset contains:
- OrderID
- Product
- Category
- Sales
- Profit
- Date

---

##  Key Insights
- Top performing products by sales
- Total revenue and profit calculation
- Sales trend over time
- Category-wise performance comparison
